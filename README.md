# Quynhanhvanan

Bản này đã được tối ưu để dễ chỉnh sửa nội dung thiệp cưới và thay ảnh nhanh.

## File chính
- `www.ewedinvite.site/AnhTuvaDieuNhi.html`

## Cách chỉnh nhanh
1. Mở file HTML bằng trình duyệt.
2. Kéo xuống phần **Khu vực chỉnh sửa nhanh**.
3. Sửa tên, ngày, địa điểm, lời mời.
4. Thay ảnh bằng 1 trong 2 cách:
   - **Upload trực tiếp** bằng các ô file.
   - Dán **URL/local path** ảnh vào các ô `URL ảnh ...` (ví dụ từ thư mục `content.pancake.vn/1`).
5. Nhấn **Lưu thay đổi** để lưu trên trình duyệt (LocalStorage).
6. Nhấn **Đặt lại mặc định** nếu muốn quay lại bản gốc.

## Nếu bạn có thư mục `content.pancake.vn/1`
- Có thể dùng nút **Áp dụng nhanh thư mục content.pancake.vn/1**.
- Mặc định nút này sẽ map:
  - `../content.pancake.vn/1/hero.jpg`
  - `../content.pancake.vn/1/gallery-1.jpg`
  - `../content.pancake.vn/1/gallery-2.jpg`
  - `../content.pancake.vn/1/gallery-3.jpg`
- Nếu tên file khác, chỉ cần sửa lại ở các ô `URL ảnh ...` rồi bấm lưu.
