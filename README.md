# Quynhanhvanan

Bản này đã được tối ưu lại để dễ chỉnh sửa nội dung thiệp cưới và thay ảnh trực tiếp.

## File chính
- `www.ewedinvite.site/AnhTuvaDieuNhi.html`

## Cách chỉnh nhanh
1. Mở file HTML bằng trình duyệt.
2. Kéo xuống phần **Khu vực chỉnh sửa nhanh**.
3. Sửa tên, ngày, địa điểm, lời mời.
4. Chọn ảnh mới ở các ô upload.
5. Nhấn **Lưu thay đổi** để lưu trên trình duyệt (LocalStorage).
6. Nhấn **Đặt lại mặc định** nếu muốn quay lại bản gốc.

## Gợi ý upload ảnh
- Ảnh bìa: tỉ lệ ngang (16:9 hoặc gần tương tự).
- Ảnh album: ảnh vuông để hiển thị đẹp nhất.
- Dung lượng nên < 2MB/ảnh để tải nhanh.
